# Ideias AGES
> Plataforma de gestão de ideias para projetos na Agência Experimental de Engenharia de Software  

Aplicação com objetivo de submeter ideias de projetos para serem executadas pelos alunos da universidade.


## Stack
- Java 8
- JSP
- Tomcat8
- MySql
